# Food Not Frenzy

The application behind the ever-growing-in-popularity foodnotfrenzy.com

## Setup

This runs on ddev, it is easy to spin up. 

### Prereqs

* install ddev

### Install

* git clone the repo
* cd into it
* ddev start

### Deployment

* All deployments are via fortrabbit. Use `ddev craft copy/code/up`
